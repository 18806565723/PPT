今天小课堂分享的内容主要是讲，Grid system（即网格系统）中row container 和 col 是什么个关系？

1.背景介绍

Bootstrap，来自 Twitter，是目前很受欢迎的前端框架。 它是一个CSS/HTML框架。由动态CSS语言Less写成。Bootstrap一经推出后颇受欢迎，一直是GitHub上的热门开源项目。

2.知识剖析

container row 和 col 分别是什么属性？

.container 类用于固定宽度并支持响应式布局的容器。

.row 和 col 类 配合用于为其赋予合适的排列和内补。

Grid system的核心思想是将屏幕划分成相等大小的格子。Bootstrap选择默认划分成12份。

规则：1.数据行(.row)必须包含在定宽容器（.container）或满宽容器（.container-fluid）中，以便为其赋予合适的对齐方式和内距(padding)。 2.具体内容应当放置在列容器（column）之内，而且只有列（column）才可以作为行容器(.row)的直接子元素

3.列与列之间有30px的间距。

Bootstrap 有哪些缺点？

1、Bootstrap 自带的样式权值很高，在开发中需要大量的代码用于覆盖原有样式。

2、不兼容IE8以下浏览器,有兼容需求的就不能使用bootstrap了。
