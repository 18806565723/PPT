1.背景介绍

曾经xml是互联网传输结构化数据的事实标准。web服务的第一次浪潮很大程度上都是建立在xml之上的，突出的特点是服务器与服务器之间的通信。但是不少人认为xml过于繁琐、冗长。

2006年，Douglas Crockford把JSON(JavaScript Object Notation)提交给IETF。而JSON的应用早在2001年就开始了。

2.知识剖析

JSON数据的书写格式是：名称/值对。

名称/值对组合中的名称写在前面（在双引号中），值对写在后面，中间用冒号隔开，2个组合之间用逗号隔开

其中值可以是：数字整数或浮点数、字符串（在双引号中）、逻辑值（true或false）、数组（在方括号中）、对象（在花括号中）、null

使用方式：

                    
var json = {"name":"myname","password":123456}

4.解决方案

在同一页面，json可以直接调用，使用“json的名称.需要调用的数据的名称”

                json.a

在不同页面，json可以利用web存储调用

  • 首先把json转化为字符串：JSON.stringify()
  • 利用web存储存储转换好的字符串，比如sessionStorage
  • 在另一个页面调用存储的字符串
  • 利用JSON.parse()将字符串解析为json对象
